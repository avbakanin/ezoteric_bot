# Базовые команды

## Назначение
- Обрабатывает стартовые команды и fallback-сообщения бота.
- Управляет показом главного меню и разделом «О боте».

## Основные обработчики
- `router.py::start_command` — команда `/start`, сбрасывает состояние, выводит приветствие и главное меню.
- `router.py::menu_command` — команда `/menu`, повторно открывает главное меню.
- `router.py::help_command` — команда `/help`, выводит памятку по функциям.
- `router.py::about_command` — текстовая кнопка «ℹ️ О боте».
- `router.py::unknown_message` — обработка прочих сообщений вне состояний.

## Зависимости
- Использует `app.shared.keyboards.get_main_menu_keyboard` для генерации меню.
- Сообщения берутся из `app.shared.messages.MessagesData`.
