# Аффирмации

## Назначение
- Отправляет пользователю ежедневную аффирмацию, учитывая историю последних 10 утверждений.
- Позволяет премиум-пользователям запрашивать дополнительные тексты без повторов за день.

## Основные обработчики
- `router.py::affirmation_handler` — выдаёт текущую аффирмацию, повторно не генерирует в пределах дня.
- `router.py::affirmation_new_handler` — доступен Premium: вынуждает генерацию нового текста, если пользователь хочет свежую формулировку.

## Интеграции
- `app.shared.calculations.get_affirmation` — бизнес-логика выбора текста и ведения истории в `users_data.json`.
- `app.shared.keyboards.get_affirmation_keyboard` — клавиатура с кнопками «Записать ощущение» и Premium CTA.

## Особенности Premium
- Безлимит на дополнительные аффирмации в течение дня.
- Расширенный список последних аффирмаций в сообщении.
