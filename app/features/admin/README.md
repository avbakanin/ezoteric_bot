# Администрирование

## Назначение
- Предоставляет службам и администраторам команды для управления доступом Premium.
- Позволяет переключать подписку пользователям без прямого редактирования базы.

## Основные обработчики
- `router.py::premium_admin_command` — команда `/premium`, поддерживающая действия: `on`, `off`, `toggle`, `status`, а также выбор целевого пользователя.

## Особенности
- Доступ ограничен пользователями, перечисленными в `ADMIN_USER_IDS` (`.env`).
- Команда обновляет подписку и синхронизирует поле `admin_mode` в `users_data.json`.
