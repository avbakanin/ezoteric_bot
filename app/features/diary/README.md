# Дневник наблюдений

## Назначение
- Даёт пользователям возможность вести записи, классифицировать их и возвращаться к истории.
- Ведёт категорию и дату для каждой записи, контролирует лимиты для бесплатных аккаунтов.

## Основные обработчики
- `router.py::_enter_diary` — запускает сценарий, спрашивает тему и ставит таймеры напоминаний.
- `router.py::diary_category_handler` — обрабатывает выбор темы, пропуск или выход.
- `router.py::handle_diary_observation` — сохраняет запись, добавляет метаданные и предлагает посмотреть историю.
- `router.py::diary_history_handler` — показывает последние три записи и мотивирует оформить Premium.

## UX и напоминания
- Через 5 минут бездействия отправляет мягкое напоминание.
- Через 30 минут автоматически завершает сценарий и возвращает в главное меню.

## Premium-мотиваторы
- Бесплатно доступны только три последних записи; премиум-пользователям предлагается полный архив и расширенная аналитика в еженедельном дайджесте.
